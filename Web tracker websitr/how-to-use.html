How to Use the Extension

Home How to Use Dashboard

Step 1: Install the Extension

Download and load the extension in developer mode in your browser.

Step 2: Visit Any Website

The extension automatically detects third-party trackers and background scripts.

Step 3: Open Extension Popup

View detected trackers categorized into Analytics, Advertising and Fingerprinting.

Step 4: View Detailed Report

Open the dashboard to see risk score and visualization.
