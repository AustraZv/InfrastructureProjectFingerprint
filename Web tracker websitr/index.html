Making Web Tracking Human Readable

Educational browser extension and reporting interface that visualizes how websites track and collect user data.

Project Purpose

This project provides a human-readable representation of web tracking mechanisms including third-party scripts, fingerprinting, analytics, and advertising trackers.

Project Scope

  • Educational visualization of tracking mechanisms
  • Tracker categorization (Analytics, Ads, Fingerprinting)
  • Risk score calculation
  • Open-source GPLv3 approach

Project Phases

Research → System Design → Detection → Classification → Extension Development → Reporting Interface → Evaluation
